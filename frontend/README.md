# Frontend (Next.js)

Next.js App Router frontend with Tailwind CSS.

## Pages

- `/` → `Frontend running`
- `/test` → fetches backend health endpoint in a Server Component

## Environment

Create `.env.local` (optional):

```env
BACKEND_URL=http://localhost:5000
```

If not set, defaults to `http://localhost:5000`.

## Run locally

```bash
npm install
npm run dev
```

App URL: `http://localhost:3000`

# Setup Notes (Windows + GitHub Desktop)

## 1) Clone and open in GitHub Desktop
- Clone repository.
- Open in VS Code from GitHub Desktop.

## 2) Run backend
```bash
cd backend
npm install
npm run dev
```

## 3) Run frontend
Open a second terminal:
```bash
cd frontend
npm install
npm run dev
```

## 4) Verify
- Frontend: `http://localhost:3000`
- API: `http://localhost:5000/api/health`
- Test page: `http://localhost:3000/test`
